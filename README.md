# Генератор мемов (l2-wb)

**Для локального запуска проекта необходимо выполнить следующие команды в терминале:**
```
git clone https://github.com/sazonov-dev/meme-generator
```
```
cd meme-generator
```
```
npm i
```
```
npm run dev
```
